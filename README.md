# eCgiLib

**eCgiLib** (easy CGI Library) is a simple to use C CGI library for developing web applications. It transparently supports the CGI methods GET and POST, as well as multipart/form-data file uploads. The library also includes tools for HTML templates and fast memory-based file handling.

Originally developed around 2000 and derived from cgic 0.5, eCgiLib was created to provide an easy-to-use interface for CGI programming in C with full file upload support.

## Why revive this project?

After 25 years, this project is still circulating on the internet but has lacked an official home for decades. As the original author, I've decided to revive and maintain it on GitHub to preserve it for future use and ensure it remains accessible and documented.
